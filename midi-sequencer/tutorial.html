MIDI connections

MIDI inputs

MIDI Inputs mapping

Inputs

Input MIDI channel events are mapped on track. Events send on channel 1 are send to track 1, events send on channel 2 are send to track 2,...

Except for channel 10 events. If there is no track number 10, events are send to the first drum track found.

Output

While playing a sequence, channel events are send to all the MIDI outputs on the channel defined by the track.

MIDI through

All input MIDI channel events are mapped to the channel defined by the track, then send to all the MIDI outputs.

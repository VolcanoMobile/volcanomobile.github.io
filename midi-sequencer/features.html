User manual

Learn how to use MIDI sequencer interface


The main feature of MIDI sequencer is to create multi-tracks sequences and loop/repeat them


Sequences

  • Custom time signature
  • Custom number of measures

Tracks

  • Mutli-tracks, each having its own instruments
  • Solo/mute tracks while composing

Patterns

  • Import from patterns catalog
  • Reuse pattern in other sequences and tracks

Controlers

  • Notes pads controler
  • Scales filtering
  • Drum pads controler for drum tracks
  • Editable drum map
  • Quantization
  • Arpegiator
  • Slide mode

Mixer

  • Mix tracks volumes
  • Easy switch between sequences
  • Control faders with and external MIDI controler

MIDI Connections

  • Connect external MIDI controler (INPUTS)
  • Connect external MIDI sound modules (OUTPUTS)

Miscellanous

  • Metronome
